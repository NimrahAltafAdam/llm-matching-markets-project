# LLM Matching Markets Project

This project evaluates how Large Language Models (LLMs) handle matching market problems with ranked preferences.

It is based on the paper:
"Matching Markets Meet LLMs: Algorithmic Reasoning with Ranked Preferences"

## Structure

- `llm_matching_markets_project.ipynb` → Final report notebook
- `data/` → Sample and processed data
- `src/` → Scripts adapted from original repo
- `outputs/` → Model outputs and evaluation results

## Note

This project selectively adapts parts of the original implementation.
